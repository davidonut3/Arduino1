Stap 2: Gebruik het blink project om te kijken of de arduino werkt.

	Bekijk de code en probeer te begrijpen hoe de code werkt.
	Kun je het LEDje elke 2 seconden laten veranderen ipv 1 seconde?

Stap 3: Sluit de buzzer aan op de arduino, één draad op een GND, en de andere op een pin, bij voorkeur A7.

Stap 5: Sluit de knoppen aan op de arduino, zie hiervoor 'pin layout knoppenprintje.txt'.

Stap 6: Gebruik het buttons project om het LEDje en/of de buzzer aan te sturen.

	Bekijk de code en probeer te begrijpen hoe de code werkt.
	Kun je een mini-orgel maken?
	Kun je de volgende game maken: 

	Het LEDje begint op z'n felst, en wordt steeds minder fel.
	Als je geen knop indrukt voordat het LEDje uit staat, verlies je.
	Als je wel een knop indrukt, krijg je meer punten naarmate je dichter bij nul zit.

Stap 7: Sluit de LED matrix aan, zie 'https://www.makerguides.com/max7219-led-dot-matrix-display-arduino-tutorial/'.

Stap 8: Gebruik het matrix project om text en sprites te tonen.

	Bekijk de code en probeer te begrijpen hoe de code werkt.
	Kun je de tekst van een knop laten zien als die knop wordt ingedrukt?

Stap 9: Gebruik de knoppen, de buzzer en de matrix samen in één project (zie project game).
